1- Instalar mpm install -g expo-cli
2- Iniciar expo init (nombre del proyecto) pero desde la version 48 pafece que ahora se inicia con este comando:
  npx create-expo-app nombreDelProyecto
3- nos pregunta como queremos iniciar el proyecto y le colocamos blank
4- Usaremos firebase de google para tener un backend, una base de datos ya creada, una 
manera de autenticarse.
5- Ejecutar expo start y si no funciona usar npm start y deberia inciar en la web un localhost
 pero en mi caso no me deja y me da opciones como estas:

8- instalar REACT Navigation e instalar las dependencias para ejecutarlo con la EXPO cli
9- importar en APP.js el NavigationContainer
10- crear una funcion MyStack para tener un orde con las multiples ventanas o screens de nuestra app pero esos componentes que se crean, vienen de React Navigation Stack para es se debe instalar la dependencia.
11- instalar ReactNavigationStack
12- Despues de instalar la dependencia, se debe importar el modulo en app.js createStakNavigator
13- llamar a la funcion createStakNavigator()
14 despues de haber creado la funcion (dentro de una constante llamada Stack) debo importar el componente en la funcion MyStack, el Stack.Navigator y por cada pantalla que quiera crear, debo crear un componente llamado Stack.Screen y creamos 3 pantallas.
