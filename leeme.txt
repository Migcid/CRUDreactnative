### COMENZAR UN PROYECTO CON REACT Y EXPO

Esta app es creada siguiendo los pasos de este video, tambien se aplican algunas mejoras, no es seguido al 100% debido que existen nuevas actualizaciones para las dependencies de npm usadas a la fecha en que se hizo el tutorial-video.

1- Instalar mpm install -g expo-cli
2- Iniciar expo init (nombre del proyecto)
3- nos pregunta como queremos iniciar el proyecto y le colocamos blank
4- Usaremos firebase de google para tener un backend, una base de datos ya creada, una 
manera de autenticarse.
5- Ejecutar expo start y si no funciona usar npm start y deberia inciar en la web un localhost
 pero en mi caso no me deja y me da opciones como estas:
	Logs for your project will appear below. Press Ctrl+C to exit.

required dependencies installed.

Please install react-native-web@~0.18.10, react-dom@18.2.0,       
@expo/webpack-config@^18.0.1 by running:

npx expo install react-native-web@~0.18.10 react-dom@18.2.0       
@expo/webpack-config@^18.0.1

If you're not using web, please ensure you remove the "web" string
from the platforms array in the project Expo config.
